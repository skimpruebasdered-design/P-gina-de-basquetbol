Acerca de

Hola, mi nombre es Angel Gabriel Pool Balam y tengo 17 años. Creé este sitio web para la escuela como proyecto de desarrollo web. Mi objetivo es compartir información sobre el básquetbol, incluyendo jugadores, equipos e historia, de forma clara y visual.

Este proyecto está diseñado para ser responsive, es decir, se adapta tanto a computadoras como a celulares y tablets. Utilicé HTML, CSS y un poco de JavaScript para los acordeones de información, con la intención de que sea interactivo y educativo.

Espero que al navegar por esta página aprendas más sobre el básquetbol y disfrutes viendo la información de manera organizada y atractiva. Este proyecto refleja mi interés en la programación web y mi pasión por el deporte.

Proyecto escolar de Angel Gabriel Pool Balam, 17 años
